# QR Generator

A simple web app that generates QR codes from web links. Enter a URL, generate a QR code, and download it as a PNG.

## Features

- Paste or type any web link
- Instant QR code generation
- Download as PNG
- Clean black & white UI
- No external APIs — runs entirely client-side

## Run locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
